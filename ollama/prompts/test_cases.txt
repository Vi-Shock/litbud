## Pass Criteria
- TEST 1: response contains celebration language. tool_calls includes track_progress with accuracy_percent=100.0 and struggled_words=[]. No get_hint.
- TEST 2: response contains phonics hint for "elephant". tool_calls includes get_hint with word="elephant".
- TEST 3: response gives hint for ONE word only (the worst). tool_calls includes all 3 in struggled_words.
- TEST 4: coaching text is entirely in Hindi (Devanagari). tool_calls are present.
- TEST 5: response contains valid JSON with tool_calls array. All three tools fire: track_progress, log_session, get_hint.
